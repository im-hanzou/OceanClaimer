# Requirement
- NodeJs v18 or latest

# How To use
- ```git clone https://github.com/downygg/OceanClaimer.git```
- ```cd OceanClaimer```
- ```npm i```
- ```set your mnemonic on mnemonic.json```
- ```npm start```
